About

I deliver enterprise-grade software and integration solutions with a strong commercial background, spanning e-commerce, research, and healthcare. My work includes leading Java/Spring services for UK Biobank (access management for 500k+ participants), architecting large-scale e-commerce platforms at RS Components, and delivering secure NHS integrations and automated data pipelines adopted by multiple trusts.

  • Integrations: HL7, FHIR, NHS Spine, Nervecentre; REST/SOAP service meshes and event-driven flows (Kafka).
  • Platforms: Developer platforms and CI/CD for teams; reproducible environments with IaC (CloudFormation/SAM/Terraform).
  • Runtime & Security: Java 17+, Spring Boot, OAuth2/OpenID Connect, auditing, secrets management, zero-downtime releases.
  • Cloud & Data: AWS (EC2/ECS/Fargate/S3/CloudFront), GitHub Actions CI/CD; PostgreSQL, Redis, S3 data lakes, scheduled ETL/PLCM exports.
  • Commercial track record: RS Components (e-commerce), University of Oxford (clinical trials), UHL (genomics & PLCM), UK Biobank (research).
